AI-Based Signal Strength Classifier:

Overview:
This project uses Machine Learning to classify signal strength into Weak, Medium, and Strong instead of using fixed threshold values.

Components Used:
Arduino UNO
Potentiometer
Python (Scikit-learn)

Methodology:
Analog signal is read using Arduino
Data is sent to PC via Serial Communication
A Decision Tree model classifies signal strength
Output is displayed in real-time

Results:
The system successfully classifies signal strength into:
Weak
Medium
Strong

Future Scope:
Integration with real RF modules
Use in wireless communication systems
Real-time signal monitoring
